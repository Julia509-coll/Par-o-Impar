📂 Contenido del Repositorio
1. Juego de Adivinar el Número (adivinar_numero.py)
Un juego interactivo donde el ordenador genera un número secreto y el usuario debe encontrarlo.


Funcionalidad: El programa piensa un número entre 1 y 100.


Interacción: Da pistas de "Más alto" o "Más bajo" en cada intento.


Puntuación: Al acertar, muestra el número total de intentos realizados.

2. Comprobador de Par o Impar (Par o Impar)
Un script sencillo para entender el uso de operadores matemáticos.


Lógica: Solicita un número al usuario y utiliza el operador de módulo (%) para verificar si el resto de la división por 2 es cero.


Resultado: Imprime en pantalla si el número es PAR o IMPAR.

3. Documentación de Desarrollo (numero_random.md)
Este archivo contiene la planificación detallada del juego de adivinanza.


Planificación: Incluye la lista de tareas necesarias para construir el juego desde cero.


Arquitectura: Define las variables necesarias (enteros, strings, contadores) y las estructuras de control como while True para los intentos.


Pseudocódigo: Una guía paso a paso que traduce la lógica del juego antes de ser programada.

🛠️ Tecnologías utilizadas
Lenguaje: Python

Entorno: Visual Studio


Librerías: random (para la generación de números aleatorios).
